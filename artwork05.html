Glicth Art: Sonny in San Francisco

Sonny in San Francisco is a glitch art piece created with audacity. I turned the image into raw data to run it through audacity, and then turned it back into an image file afterwards to create a distorted version of this once colorful image of a good day. It is meant to be a representation of how a good memory can overtime become faded, as we focus less on the positives and joys of life at times.
